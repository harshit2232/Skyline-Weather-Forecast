# Skyline-Weather-Forecast
A hand-crafted, single-file weather forecasting app with an hour-by-hour view, live city search, and a dark, editorial-style UI — no build tools, no API key, no backend required.
Feature
1.City search with autocomplete — type any city and get instant suggestions, or use your current location
2.Hour-by-hour forecast — scroll through the next 24 hours, click any hour to see its full detail (temperature, humidity, wind, pressure, visibility)
3.Current conditions — temperature, feels-like, and condition icon in a dynamic hero that shifts color for day, dusk, and night
4.Key metrics at a glance — UV index, humidity, wind speed and direction, air pressure, visibility, and precipitation chance
5.7-day outlook with a min/max range bar for each day
6.°C / °F toggle — switch units instantly across the whole app
7.Polished dark UI — glassmorphism, gradient accents, smooth animations, and a custom logo/favicon
8.Fully responsive and accessible (keyboard navigation, ARIA roles, reduced-motion support)
